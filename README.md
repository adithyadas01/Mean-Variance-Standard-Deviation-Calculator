# Mean-Variance-Standard-Deviation-Calculator
Mean-Variance-Standard Deviation Calculator is a Python project that uses NumPy to perform mathematical calculations on a 3×3 matrix. It takes a list of 9 numbers, reshapes them into a 3×3 array, and calculates the following for rows, columns, and the entire matrix:  Mean  Variance  Standard Deviation  Maximum  Minimum  Sum  .
